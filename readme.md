# 初回実行
- このリポジトリをクローンした際に1回だけ実行してください．
- それ以降は実行しなくても大丈夫です．

```shell
$ sh entrypoint.sh
```

- また，エラー内容は `question_description.md` を参照してください．
> note
> ######################################################################
> 
> 絶対に，`git add`と`git commit`のコマンドは実行しないでください！！
> 
> ######################################################################
